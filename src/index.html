개발 스택

웹 개발의 기초

  HTML, CSS, JAVASCRIPT는 웹 개발의 기초입니다. 프레임워크 없이 Plain Javascript를 이용하여 개발한 경험은 웹의 기본 지식을 익히는데 큰 도움이 되었습니다.

한 걸음 더

  기존의 느슨한 타입을 가지고 있는 Javascript를 Typescript로 대체했습니다. 현재는 Node JS, React JS를 포함한 모든 Javascript 프로젝트를 Typescript를 이용해서 구현합니다.

Framework & Runtime

  Node JS는 Javsacript를 브라우저 밖에서도 구동시켜줍니다. npm모듈과 각종 프레임워크를 사용해 Frontend, Backend를 구현해왔습니다.

  React를 사용하여 Frontend를 개발했습니다. React 개발을 Functional Programming으로 가능케 하는 Hooks API를 적극적으로 사용하여, 유동적인 웹 어플리케이션을 빌드하는데 React를 사용했습니다.

  Nest JS를 이용하여 Backend를 개발했습니다. Express JS와 대비되는 강제되는 Module 구조를 통해 빌드하며 코드의 재사용성에 대해 고민하고, 직관적이며 Error를 덜 야기하는 어플리케이션을 개발하려 노력했습니다.

Server & Database

  Google이 개발한 Serverless 기반의 Firebase app은 다양한 Backend API를 제공합니다. 이를 통해 Web Hosting, NOSQL-DB, Serverless Functions 로 구성되어 있는 웹 어플리케이션을 빌드했습니다.

  Naver Cloud Platform 서비스를 사용하여 Linux Ubuntu 서버를 호스팅 중입니다. 해당 서버에 Nest JS와 MYSQL을 구동시켜 웹 어플리케이션 Backend를 호스팅했습니다.

  Docker Container를 통해 Server를 관리했습니다. docker-compose 등의 기능을 통해 개발 환경에서의 테스트 및 서버로의 배포까지 전반적 과정을 Docker를 통해 개발합니다.

  SQL Database, NOSQL Database 기반인 Mysql, mongodb 를 사용하여 Database를 구축, Backend Server와 연결하여 개발했습니다.
